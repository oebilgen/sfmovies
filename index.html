Welcome to San Francisco Movies.

This project helps you see the movies that were filmed in San Francisco on a map. If you'd like to get the source code, follow the instructions:

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

How to use it?

Visit www.sfmovies.org, where you will see San Francisco movies on a map. You can zoom in and out, click the movie markers to see detailed information about a movie, watch it online or purchase its DVD on Amazon, among many other things. Curious what movie was shot in your neighborhood or near your favorite dive bar? Visit www.sfmovies.org now! :)

The architecture

This section is interesting for those who wants to learn how this all work.

SFMovies consists of 3 pieces:

  • Web frontend: That's where you see the map.
  • REST API: Provides data for the website to display the movies.
  • CDN storage: Contains the movie data and the files needed for the webiste.
